# Changelog

## 1.0.0

Initial public release of Nest, a Finder-native AI agent for macOS.

- Finder-native AI agent experience
- Natural-language file understanding and actions
- Bring-your-own AI provider settings
- Gemini, OpenRouter, OpenAI-compatible, and Ollama support
- Auto-run settings for harmless commands
- Command preview for risky actions
- In-bar progress and result display
- Activity log window
- First-run onboarding
- Permission health checks
- Optional ImageMagick support
- Standalone `swiftc` build script
